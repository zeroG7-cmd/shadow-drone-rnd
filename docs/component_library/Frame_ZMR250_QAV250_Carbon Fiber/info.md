# Frame: ZMR250 / QAV250 Carbon Fiber Frame

## Category
- Structural / Frame

## Overview
250mm class carbon fiber quadcopter frame designed for FPV racing and freestyle builds.

## Specifications
- Wheelbase: 250mm
- Frame Type: X configuration
- Material: Carbon fiber
- Arm Thickness: 3mm–4mm
- Plate Thickness: ~1.5mm

## Physical
- Mass: ~141g
- Dimensions: ~250mm diagonal, ~80mm height

## Compatibility
- Motors: 2204–2208 (e.g. 2205)
- Propellers: 5 inch
- Battery: 3S–4S (1300–1800mAh typical)

## System Role
- Main structural base
- Mounts all components (motors, ESCs, stack, battery)

## Integration
- Motors mounted on arms
- ESCs mounted on arms (current setup)
- Stack mounted at center
- Battery mounted below (modified)
