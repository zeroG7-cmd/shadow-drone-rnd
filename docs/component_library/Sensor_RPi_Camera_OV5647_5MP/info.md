# Component: Raspberry Pi Camera (OV5647 5MP)

## Category
Sensor (Vision)

## Overview
5MP camera module for Raspberry Pi using OV5647 sensor.
Supports 1080p video and CSI interface.

## Physical
- Dimensions: ~25mm x 24mm
- Mounting: 4 holes (corner)
- Connector: 15-pin CSI ribbon
- Lens: fixed / adjustable focus module

## Performance
- Resolution: 5MP
- Video: 1080p
- Sensor: OV5647

## System Role
Provides visual input for Shadow drone (future computer vision / FPV)

## Notes
- Likely third-party module (not official Raspberry Pi V2/V3)
- Night vision capability depends on IR configuration
