# Sensor: RPLIDAR A2M12

## Type
- 2D LiDAR (laser scanner)

## Specifications
- Type: 2D LiDAR
- Range: up to 12m
- Scan: 360°
- Frequency: ~10 Hz

## Electrical
- Voltage: 5V
- Interface: USB / UART

## Physical
- Mass: ~190g
- Dimensions: ~70mm diameter x 41mm height

## Output
- Distance + angle data points

## Use Case
- SLAM (mapping)
- obstacle detection
- indoor navigation
