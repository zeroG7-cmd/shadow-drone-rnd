# ESC: XXD 30A XT60 Brushless ESC

## Specifications
- Continuous current: 30A
- Burst current: ~40A
- Battery: 2S–4S LiPo
- BEC output: 5V / ~2A

## Type
- Sensorless brushless ESC
- Individual (one per motor)

## Physical
- Mass: ~7g (each)
- Dimensions: ~55 x 25 x 10 mm


## Connections
- Input: XT60 battery / PDB
- Output: 3-phase motor wires
- Signal: PWM / DSHOT (depending on FC)

## System Role
- Controls motor speed and direction
