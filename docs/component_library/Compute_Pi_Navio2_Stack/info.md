## Assembly: Raspberry Pi 3B + Navio 2 Stack

## Overview
Stacked configuration of Raspberry Pi 3B and Navio 2 housed in a 3D printed enclosure.

## Physical
- Mass: ~120–150g (estimated combined)
- Dimensions: ~90 x 65 x 40 mm (including case)

## Mounting
- Fixed to drone frame
- Enclosed in 3D printed case

## System Role
- Central compute + flight control unit

## Integration
- Connected to ESCs (control signals)
- Connected to sensors (LiDAR, camera)
